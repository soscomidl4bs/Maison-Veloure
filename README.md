## Bakery Website UI + Dashboard

Website bakery dengan desain modern dan elegan, dilengkapi dengan halaman autentikasi (login & register) serta dashboard. Dibangun menggunakan HTML, CSS, dan JavaScript dengan tampilan clean, aesthetic, dan user-friendly.

## Preview

Tampilan mengusung konsep **elegan & premium bakery (French-style)** dengan warna soft dan layout modern.

Halaman yang tersedia:
- Welcome Page (Landing)
- Login Page
- Register Page
- Dashboard

  
## Fitur Utama

- UI modern dan elegan
- Responsive design  
- Halaman autentikasi (Login & Register)  
- Dashboard  
- Clean & structured code  
- Mudah dikustomisasi  

## Teknologi

- HTML  
- CSS  
- JavaScript

## Struktur Project

- welcome.html → Welcome / Landing Page  
- login.html → Halaman Login  
- register.html → Halaman Register  
- index.html → Halaman Dashboard

   git clone https://github.com/soscomidl4bs/Maison-Veloure.git
